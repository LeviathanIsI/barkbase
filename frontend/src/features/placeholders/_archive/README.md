# Archived Placeholder Components

## ⚠️ DO NOT IMPORT FROM THIS FOLDER

This folder contains **57 archived placeholder components** and **13 placeholder routes** that were created during early development but are **not currently in use**.

## Purpose

These components represent early prototypes and feature concepts that were:
- Designed but never fully implemented
- Replaced by different implementations
- Deferred to future development cycles

## Contents

### Components (57 files)
UI components for features that were planned but not completed:
- Payment and billing dashboards
- Advanced analytics views
- Staff management tools
- Check-in/check-out modals
- Package management
- Report builders
- And more...

### Routes (13 files)
Route placeholder pages that were never wired into the router:
- DaycareCheckin
- Facilities overviews
- Package management
- Team management
- And more...

## Status

**Archived** - Not part of any active build or deployment.

These files are preserved for:
1. Historical reference
2. Potential future reactivation
3. Design pattern documentation

## Do NOT

- Import any component from this folder
- Add new components to this folder
- Reference these in the router
- Include in any build configuration

## Reactivation

If a feature needs to be reactivated:
1. Move the component out of `_archive/`
2. Update imports and integrate with current architecture
3. Test thoroughly before deployment
4. Remove from this archive
